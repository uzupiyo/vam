# RIN SURVIVORS - All Assets Integrated Test

現時点の素材を一旦すべてゲームに組み込んだ確認版です。

## 入っているもの

- タイトル画面
- キャラクター選択画面
- Rin 通常ドット
- Rin 立ち絵
- 敵4種
- 背景
- 最新アイテム3種（緑背景クロマキー済み）
- Rin武器アイコン
- 炎・雷・銃撃・取得エフェクト

## 起動

`index.html` を開いてください。

## 確認用ページ

`asset_preview.html` で全素材のパスと透過状態を確認できます。

## テスト操作

ゲーム中:

- `T`: アイテム3種をプレイヤー近くに出す
- `Y`: 炎・雷エフェクトを出す
- `Space`: 一時停止
