# All Assets Integrated

この版は、現時点で使っている素材をゲームに組み込んだ確認版です。

## 組み込み済み

- Rin 通常 idle
- Rin 立ち絵
- 敵4種: slime / bat / golem / boss
- 背景
- UI
- キャラクター選択画面
- 炎・雷・銃撃・アイテム取得エフェクト
- 最新の緑背景クロマキー済みアイテム
  - magnet
  - barrier_shield
  - heal_potion（ゲーム内では heal_cross.png として利用）
- Rin dual pistols icon

## 確認用操作

ゲーム中に以下を押すと確認できます。

- `T`: 近くにアイテム3種を出す
- `Y`: 炎・雷エフェクトを出す
